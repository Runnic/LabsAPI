Labs API

Projeto desenvolvido para seleção da WA Project.

O projeto consiste em uma restAPI que cadastre laboratórios e exames em lote e relacione esses dois se necessário.

Desenvolvi esse projeto utilizando Typescript com o serviço em nuvem de banco de dados MongoDB Atlas e arquitetura DDD (Domain-Driven Design).

Também utilizei o ESLint e Prettier para manter uma boa organização de código.

**Obs: A URL base da API é a mesma deste site.

Para mais informações acesse o repositorio no GitHub.
