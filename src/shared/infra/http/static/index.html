Labs API

Projeto desenvolvido para seleção da WA Project.

O projeto consiste em criar uma restAPI que cadastre laboratórios e exames em lote e relacione esses dois se necessário.

Desenvolvi esse projeto utilizando Typescript com Express.Js e arquitetura DDD (Domain-Driven Design).

Também utilizei o ESLint e Prettier para manter uma boa organização de código.

**Obs: A URL base da API é a mesma deste site.
